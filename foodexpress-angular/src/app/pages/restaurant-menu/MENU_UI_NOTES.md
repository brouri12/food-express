# Menu UI Notes

Ce document décrit les interactions principales de la page menu:

- Filtres: recherche, disponibilité, végétarien, populaire, prix max.
- Tri: défaut, nom, prix croissant/décroissant.
- Pagination: blocs de catégories affichés par pages.
- Quantité: stepper par plat avant ajout panier.
- Cache: affichage diagnostic + action de purge manuelle.

Objectif: garder une UX rapide même sans backend stable.
