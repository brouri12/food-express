# 🍽️ FoodExpress - Guide Utilisateur

Bienvenue sur **FoodExpress**, votre plateforme de livraison de repas nouvelle génération !

## 🚀 Démarrage Rapide

### 1. Créer un Compte
- Visitez la page d'**inscription** (`/signup`)
- Choisissez votre type de compte :
  - 🛍️ **Client** : Commander des repas
  - 🛵 **Livreur** : Livrer des commandes
  - 🍴 **Restaurateur** : Gérer votre restaurant
- Remplissez vos informations et validez

### 2. Explorer les Restaurants
- Parcourez les **catégories de cuisine** sur la page d'accueil
- Utilisez la **barre de recherche** pour trouver un restaurant ou un plat
- Filtrez par **note**, **temps de livraison**, ou **promotions**

### 3. Commander
1. Sélectionnez un **restaurant**
2. Parcourez le **menu** par catégories
3. Ajoutez des plats à votre **panier**
4. Appliquez un **code promo** si disponible
5. Choisissez votre **adresse de livraison**
6. Sélectionnez votre **moyen de paiement**
7. **Validez** votre commande

### 4. Suivre votre Livraison
- Accédez à la page de **suivi** (`/delivery/:orderId`)
- Visualisez la **position du livreur** en temps réel sur la carte
- Consultez le **temps d'arrivée estimé**
- **Contactez le livreur** si nécessaire

### 5. Évaluer votre Commande
- Après livraison, vous recevrez une **notification**
- Donnez votre **note** (1-5 étoiles)
- Laissez un **commentaire**
- Ajoutez des **photos** de vos plats (optionnel)

## 💡 Astuces & Conseils

### 🏆 Optimiser vos Commandes
1. **Commandez pendant les heures creuses** (14h-17h) pour une livraison plus rapide
2. **Groupez vos commandes** pour atteindre le minimum et éviter les frais
3. **Abonnez-vous aux notifications** pour ne rater aucune promo
4. **Ajoutez plusieurs adresses** (maison, bureau) pour faciliter vos commandes

### 💰 Économiser
- Consultez la section **Offres Spéciales** sur la page d'accueil
- Utilisez les **codes promo** lors du checkout
- Cherchez le badge **"Livraison gratuite"** sur les restaurants
- Commandez dans les restaurants **"Promus"** pour des réductions

### 📱 Navigation Mobile
- **Bottom Bar** : Accès rapide aux sections principales
- **Swipe** : Faites glisser pour explorer les catégories
- **Tap** : Appuyez sur les notifications pour plus de détails

## 🛠️ Résolution de Problèmes

### ❌ Problème de Connexion
1. Vérifiez votre **email** et **mot de passe**
2. Utilisez "**Mot de passe oublié**" si nécessaire
3. Essayez de vous connecter avec **Google** ou **Apple**

### 🛒 Le Panier ne s'enregistre pas
- Assurez-vous que les **cookies** sont activés
- Videz le **cache** de votre navigateur
- Essayez en **navigation privée** pour tester

### 🚫 Restaurant Indisponible
- Vérifiez les **horaires d'ouverture**
- Certains restaurants ont des **jours de fermeture**
- Essayez de **commander plus tard**

### 📍 Problème d'Adresse
- Vérifiez que votre **adresse est complète** (numéro, rue, code postal, ville)
- Certaines zones peuvent être **hors de portée** de livraison
- Contactez le **support** pour vérifier la disponibilité
